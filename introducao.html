Introdução ao HTML

TAGS - Estruturais
<html>...</html> A tag HTML, abreviação de HyperText Markup Language, é usada para estruturar o conteúdo de uma página da web.
<head>...</head> Contém metadados sobre o documento HTML, como o título da página e links para estilos e scripts.
<title>...</title> Define o título da página, exibido na barra de título do navegador e em resultados de pesquisa.
<body>...</body> Envolve todo o conteúdo visível da página.
<main>...</main> Identifica o conteúdo principal da página.

TAGS - Listas
<ul>...</ul> Cria listas não ordenadas, uma lista de itens onde a ordem dos itens não é importante
<ol>...<ol> Cria uma lista ordenada, onde os itens são numerados por padrão, mas podem ser marcados de outras formas.
<li>...</li> Define um item em uma lista (<ul> ou <ol>). Cada item de lista deve ser envolvido nesta tag.

TAGS - Tags Semânticas Descrição
<header> Define o cabeçalho de um documento ou uma seção
<nav> Define uma seção de navegação do documento
<main> Define o conteúdo principal de um documento
<aside> Define conteúdo lateral relacionado ao conteúdo principal
<article> Define um conteúdo independente que pode ser republicado separadamente
<section> Define uma seção genérica em um documento
<footer> Define o rodapé de um documento ou uma seção

TAGS - Tags de Texto
<p>...</p> Define um parágrafo de texto.
<h1>...<h6> Define cabeçalhos de diferentes níveis, do mais importante (h1) ao menos importante (h6)
<b>...</b> Define texto em negrito.
<i>...<i> Define texto em itálico.
<u>...</u> Define texto sublinhado.
<s>...</s> Define texto riscado.
<sup>...</sup> Define texto sobrescrito.
<sub>...</sub> Define texto subscrito.

TAGS - Tags de Imagem
<img> Define uma imagem
<figure> Define uma figura com legenda opcional
<figcaption> Define a legenda para uma figura
<map> Define um mapa de imagem
<area> Define uma área em um mapa de imagem

TAGS - Tags de Links/Âncoras Descrição
<a> Define um hiperlink
<link> Define a relação entre o documento atual e um recurso externo (como um estilo CSS)
<nav> Define uma seção de navegação do documento
<button> Define um botão clicável
<form> Define um formulário HTML para entrada de usuário
